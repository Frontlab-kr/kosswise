SBOM Import
※ SBOM 파일을 특정 프로젝트/버전에 등록하여 식별 작업을 진행할 수 있습니다.
※ SBOM 파일은 1개의 프로젝트/버전에만 Import 가능합니다(복수 등록 x)
※ 기존 Import 했던 SBOM 파일이라면, Import 당시 선택한 프로젝트/버전에만 재등록 가능합니다.
※ BDIO 파일 Import는 파일 정보로 프로젝트와 버전이 생성되므로 프로젝트 및 버전 지정이 불가능합니다.
